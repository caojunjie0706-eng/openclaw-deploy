# Seth Godin：如何传播你的想法

## 核心观点
想法的传播不再依赖于大声疾呼，而是依赖于“值得传播”（Remarkable）。在信息过载的时代，平庸即隐形。你需要创造出能够引起特定人群共鸣并促使他们主动分享的东西。

## 关键增长方法论
1.  **寻找最小可行市场 (Smallest Viable Market)**：
    *   不要试图取悦所有人。找到那一小群深爱你的产品或想法的人，让他们成为你的传播者（Sneezers）。
2.  **创造“紫牛” (Be Remarkable)**：
    *   在黑白奶牛群中，紫牛才是值得谈论的。卓越的产品本身就是营销，而不是事后的装饰。
3.  **许可营销 (Permission Marketing)**：
    *   尊重用户的注意力。通过提供价值来换取他们进一步了解你的许可，而不是强行打扰。
4.  **建立部落 (Build a Tribe)**：
    *   人们渴望连接。品牌应该成为连接具有相似价值观的人群的纽带。

## 核心洞察
*   **安全是危险的**：在今天，走稳妥路线就是走死胡同。平庸是最大的失败。
*   **信任比流量更贵**：传播的本质是信任的传递。如果人们不信任你，再精妙的创意也无法传播。
