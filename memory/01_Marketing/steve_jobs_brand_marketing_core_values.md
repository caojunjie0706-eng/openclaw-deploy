# 乔布斯：品牌营销的核心是价值观

## 核心观点
营销不是关于参数或性能的对比，而是关于“你是谁”以及“你代表什么”。品牌在用户心中占据位置的方式，是通过共有的价值观建立灵魂层面的联结。

## 关键逻辑
1.  **超越产品性能**：在这个复杂、嘈杂的世界里，人们没有机会记住所有的规格。品牌必须清晰地告诉世界它的立场。
2.  **回归初心**：营销必须回归到品牌最核心的驱动力。例如：Apple 并不只是制造更快的电脑，而是为了那些“改变世界的人”提供工具。
3.  **价值观的共鸣**：最好的营销是找到那些相信你所相信的事的人。
4.  **明确“不做什么”**：价值观也体现在你的拒绝中。通过明确界限，品牌形象会变得更加锐利。

## 核心洞察
*   **营销的终极目标**：是让用户在选择你的产品时，感受到一种身份的认同和自豪感。
*   **品牌是承诺的集合**：而价值观是这个承诺的基石。
