# YC 创业课之销售篇

## 核心观点
对于初创企业，销售不是为了收入，而是为了“学习”。在产品未达到 PMF（产品市场匹配）之前，创始人必须亲自做销售，以便直接获取反馈并快速迭代产品。

## 关键销售策略
1.  **创始人亲自销售**：
    *   在公司规模扩大之前，不要雇佣销售副总裁。创始人最懂产品，也最能通过销售发现产品缺陷。
2.  **寻找“极客”用户 (Early Adopters)**：
    *   专注于那些忍受着巨大痛苦并愿意尝试不完美解决方案的人。
3.  **做那些“无法规模化”的事**：
    *   手写邮件、亲自上门、为每个客户做深度定制化的演示。这种亲密感是初创企业的唯一优势。
4.  **定价即筛选**：
    *   不要害怕收费。愿意付费的用户提供的反馈比免费用户更有价值。

## 核心洞察
*   **销售是产品开发的延伸**：每一次拒绝都是一次宝贵的调研机会。
*   **速度胜过一切**：在初创阶段，快速跟进一个潜在客户比完美的销售演示更重要。
