# 硅谷顶级 PR：如何塑造创始人 IP

## 核心逻辑：穿透噪音的三大抓手
1. **回归极致的人性 (Humanity)**：人连接人，机构无法连接人。不要躲在 Logo 后面，要让具体的人站出来。
2. **传染性的信念感 (Conviction)**：用 100% 的确定性对抗世界的不确定性。
3. **叙事弧线 (Narrative Arc)**：像《一千零一夜》一样，将信息模块化，保持悬念。

## 维恩图策略：寻找“精神诱饵”
- **左圆**：你想说的（产品/技术）。
- **右圆**：受众关心的（焦虑/利益）。
- **重叠区**：你的切入点（Gateway Drug）。先用他们关心的内容钩住大脑，再带入你的产品。
