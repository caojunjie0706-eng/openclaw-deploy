# 人们先感性评估，后理性解释自己的决定

## 核心观点
人类的决策过程并非纯粹理性。决策通常在潜意识层面由情绪和直觉驱动（感性评估），随后大脑会调动逻辑来为该决定寻找合理的借口（理性解释）。营销的成功在于先击中情绪，再提供逻辑。

## 关键逻辑
1.  **直觉优先 (System 1)**：大脑倾向于通过快思考进行初步筛选。如果品牌不能在第一时间通过视觉或情绪唤起正向共鸣，理性的说服很难奏效。
2.  **为理性提供借口**：一旦用户在感性上认同了产品，他们需要功能参数、性价比、耐用性等数据来支撑自己的决定，以消除“认知失调”。
3.  **情绪锚定**：品牌的力量在于创造一种“感觉”。这种感觉会成为用户记忆中的锚点。
4.  **叙事的双重作用**：好的故事既要包含感性的共鸣点，也要包含理性的事实支撑点。

## 核心洞察
*   **卖的不是参数，是状态**：用户买的是拥有产品后的某种心理状态或身份象征。
*   **不要只讲逻辑**：如果你只通过逻辑赢得了辩论，你可能还是会输掉订单，因为你没有赢得他们的心。
