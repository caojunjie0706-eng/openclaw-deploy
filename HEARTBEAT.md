# JARVIS 系统监控哨兵

## 任务清单：
1. **Token 负载检查**：运行 `session_status` 检查主会话 tokens。
2. **报警阈值**：如果 Inbound Token > 150,000 (75% 警戒位) 或 > 180,000 (90% 临界位)。
3. **报警动作**：
   - 在对话中发送：“⚠️ JJ，当前主会话 Token 已达到高负载（{current_tokens}），为了防止 API 速率受限，建议执行 /clear 操作。”
4. **静默状态**：如果一切正常，仅记录状态，不打扰用户。
